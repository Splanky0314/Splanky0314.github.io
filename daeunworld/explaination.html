Daeun World

- 사이트 소개 -

Daeun World는 DaeunKim의 2번째 Github Page로 2020.12.31에 개발이 시작되었다.

Daeun World 개발 목적

Daeun World 개발을 통해 웹 프로그래밍 능력을 기르기 위함을 첫번째 목적으로 하여 만들어졌습니다.

  1. 웹 개발을 통한 웹 프로그래밍 능력을 기르기 위함
  2. 탐구 활동의 기록을 남기는 공간으로 사용하기 위함
  3. 자기소개를 위함
  4. 방문자와의 소통을 위함

Daeun World 개발 과정

2020.12.31에 개발을 시작하였으며 현재 아직 개발 진행중입니다. 빠르게 개발을 진행하려 하다 보니 하루에 엄청난 시간을 웹 개발에 투자하고 있습니다!

  • 2020.12.31 info 사이트 개발 시작
  • 2020.01.01 본 사이트(index) 개발 시작
  • 2020.01.02 info 사이트 완성
  • 2020.01.03 사이트 소개 페이지 완성

개발자 소개

아직 개발을 배워나가고 있는 초심자입니다! 고1이 되고 한아름이라는 한민고 학생들을 위한 저는 서비스를 제공하는 단체의 개발팀으로 들어가 처음으로 코딩을 접했습니다. 그 뒤 웹 프로그래밍을 배우고 C언어 Python를 시작하며 코딩에 대한 흥미를 키워나가고 있습니다.


자세한 내용은 아래 Info사이트를 참고해주시기 바랍니다.

Daeun World의 특징

Daeun World는 저의 두번째 Github Page입니다. 따라서 첫번째로 만들었던 사이트의 개선해야 할 점과 단점을 적극 보완하여 지금의 Daeun World를 만들었습니다.

게시판별 소개

  1. Info

    2. DaeunKim의 자기소개 사이트입니다.


    Info 사이트 이동하기

  2. 사이트 소개

    3. Daeun World에 대한 소개글을 게시해 둔 공간입니다.

  3. 탐구 프로젝트

    Github의 특성 상 유료 회원이 아니면 소스 코드가 전부 공개되고, 외부인이 사이트 내용을 볼 수 있기에 민감한 자료들은 업로드 하지 않고 있습니다.

  4. 개발 일지

    5. Daeun World를 비롯 각종 개발을 하면서 느낀점과 그날의 진척 과정을 기록해 둔 공간입니다.

  5. 방명록

    6. 방문자들이 댓글 형식으로 흔적을 남길 수 있는 공간입니다.
